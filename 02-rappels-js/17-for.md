# for

Exécuter un bloc un certain nombre de fois.

```js
for (initialisation; condition; iteration) {
	// A exécuter tant que la condition est vraie
}

/*
* initialisation : est exécuté avant la première itération.
* condition : est testé à chaque itération, 
* 	on arrête la boucle si condition vaut `false`.
* exécute l'instruction iteration après chaque itération.
*/

for (i=0; i<10; i++) {
	document.write(i);	
}
```
