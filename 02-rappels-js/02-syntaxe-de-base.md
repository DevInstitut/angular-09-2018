# Syntaxe de base

Chaque ligne de code correspond à une instruction.

> Il est recommandé de séparer les instructions par des **;** (optionnel dans le langage).

```javascript
var a = "bonjour";
console.log(a);
```

Les blocs d'instructions sont placés entre **{ }**

```javascript
if (a == 1) {
	a++;
	console.log(a);
}
```
