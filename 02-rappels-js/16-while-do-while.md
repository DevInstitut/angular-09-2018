# while / do...while

Exécuter un bloc de code _tant que_ une condition est vraie.

```js
while (condition) {
	// A exécuter tant que la condition est vraie
	// jamais exécuté si la condition est
	// initialement fausse
}

do {
	// A exécuter tant que la condition est vraie
	// exécuté une fois si la condition est 
	// initialement fausse
} while (condition) ;

```
